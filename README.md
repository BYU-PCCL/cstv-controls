# @footron/web

## Installation

Update to Yarn 2 (see
[Yarn docs](https://yarnpkg.com/getting-started/install#updating-to-the-latest-versions)
for background):

```sh
yarn set version berry
```

Install packages:

```sh
yarn install
```

## Running

Start server:

```sh
yarn start
```

Build for production:

```sh
yarn build
```

Run tests:

```sh
yarn test
```

Run storybook:

```sh
yarn storybook
```
